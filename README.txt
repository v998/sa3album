[sa3album][1] 是一款運行在[Google App Engine][2]的簡易貼圖程序

The source code from the original author could be found on https://bitbucket.org/benmao/sa3album

This app has been migrated to Python 2.7 since Google App Engine has shutdown support for Python 2.5 on June 20, 2017.


**特點**

* 友好的URL
* 圖片不壓縮，無水印
* 多文件上傳
* 方便分享
* 防盜鏈
